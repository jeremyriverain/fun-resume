# Fun resume

A demo is available on [https://fun-resume.surge.sh](https://fun-resume.surge.sh).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploy
```
npm run deploy
```
